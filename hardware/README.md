# Wiring Guide
To simplify wiring we'll set a standard wire color for each functionality.

| Color  | Description      |
|--------|------------------|
| RED    | Positive voltage |
| BLACK  | Ground voltage   |
| YELLOW | SPI - CLK        |
| WHITE  | SPI - CIPO/MISO  |
| BLUE   | SPI - COPI/MOSI  |
| GREEN  | SPI - CS         |
